Vertex AI to BigQuery

Proof of Concept

Demonstration of a simple chat integration between Vertex AI and BigQuery.

This integration makes it possible to chat with your BigQuery data.

Dataset Schema

The dataset used for this proof of concept is AI generated Google Ads performance data.
Below is the schema of the dataset:

Field Name Data Type
Date DATE
CampaignID INTEGER
CampaignName STRING
AdGroup STRING
Impressions INTEGER
Clicks INTEGER
Conversions INTEGER
Cost INTEGER
